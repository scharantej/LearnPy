Here are the solutions to the exercises.

  1. python for i in range(1, 11): print(i)
  2. python def sum(a, b): return a + b print(sum(1, 2))
  3. python def factorial(n): if n == 0: return 1 else: return n * factorial(n - 1) print(factorial(5))
  4. python def reverse(string): return string[::-1] print(reverse("Hello, World!"))
  5. python def max(list): max = list[0] for i in range(1, len(list)): if list[i] > max: max = list[i] return max print(max([1, 2, 3, 4, 5]))
